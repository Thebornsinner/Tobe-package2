# tobe-package2

> to render components in the package globally

[![NPM](https://img.shields.io/npm/v/tobe-package2.svg)](https://www.npmjs.com/package/tobe-package2) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save tobe-package2
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'tobe-package2'
import 'tobe-package2/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
